# SQL Data Warehouse & Analytics Project

**End-to-End Modern Data Warehouse** built using **Medallion Architecture** (Bronze → Silver → Gold) for large-scale analytics.

![Architecture](docs/architecture-diagram.png)

## 🚀 Project Overview

This project demonstrates building a complete **SQL Data Warehouse** from raw data to analytics-ready layer. It includes full ETL pipelines, data modeling, and business intelligence queries.

### Key Features
- Implemented **Medallion Architecture** (Bronze, Silver, Gold layers)
- Built robust **ETL pipelines** for data ingestion and transformation
- Designed **Star Schema** with Fact and Dimension tables
- Created advanced analytical queries for business insights
- Applied data quality checks and best practices

## 🛠 Technologies & Skills
- **SQL Server** | Advanced SQL
- Window Functions, CTEs, Stored Procedures
- Data Modeling (Star Schema)
- ETL Development
- Data Warehousing Best Practices
